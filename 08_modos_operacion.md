1 - MODO ENCOGIDO

Al activarse este modo, el robot contrae todas sus articulaciones, volviéndose más compacto. Este modo se utiliza para cumplir con los requisitos de tamaño máximo del robot. Este modo es el inicial del robot.

2 - MODO STANDBY

Al pulsar el botón del LED rojo, estando el robot encogido, pasaremos al modo STANDBY. Este modo nos permite acceder al resto de modos. Durante este modo, el robot tiene 3 patas elevadas.

Al pulsar el botón del LED rojo desde este modo, pasamos al modo encogido. De igual manera, si pulsamos el botón amarillo haremos la Tarea 1 y con el azul la Tarea 2.

3 - TAREA 1

En este modo, el robot ejecuta la lógica de control necesaria para superar la Prueba 1.

4 - TAREA 2

En este modo, el robot ejecuta la lógica de control necesaria para superar la Prueba 2.
