## 1 - MODO ENCOGIDO

Al activarse este modo, el robot contrae todas sus articulaciones, volviéndose más compacto. Este modo se utiliza para cumplir con los requisitos de tamaño máximo del robot. Este modo es el inicial del robot. Durante este el led rojo siempre esta encendido.

## 2 - MODO STANDBY

Este modo nos permite acceder al resto de modos. Durante este modo, el robot tiene 3 patas elevadas.

Al entrar en el modo Standby veremos que los tres leds se encuentran encendidos. Si pulsamos el botón del led rojo el robot volverá al modo encogido. Si pulsamos el botón del led amarillo, el robot entrará en el modo tarea 1. Por último, si pulsamos el botón del led azul pasaremos al modo tarea 2.

## 3 - TAREA 1

En este modo, el robot ejecuta la lógica de control necesaria para superar la Prueba 1.

Durante la ejecución de la tarea, se encenderá el led amarillo. Si se vuelve a pulsar el botón del led amarillo, el robot volverá al estado Standby.

## 4 - TAREA 2 

En este modo, el robot ejecuta la lógica de control necesaria para superar la Prueba 2.

Durante esta tarea habrá el siguiente patrón de leds:

- Si detecta una pared se encenderán tanto el led rojo como el amarillo.
- Si detecta un obstaculo se encenderán tanto el led eojo como el azul.

Durante la ejecución de la tarea, se encenderá el led azul. Si se vuelve a pulsar el botón del led azul, el robot volverá al estado Standby.
